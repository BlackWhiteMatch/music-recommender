# 基于爬虫 + AI 轻量化模型的音乐推荐系统

本项目使用 Python、Flask 和 scikit-learn 实现一个简单的音乐推荐系统。
系统流程：爬虫（或模拟爬虫）生成歌曲数据 → 特征提取（TF-IDF） → 内容相似度推荐 → Web 页面展示结果。

## 环境依赖

- Python 3.10 或以上
- 依赖包见 `requirements.txt`

安装依赖（在项目根目录执行）：

`pip install -r requirements.txt`

## 运行方式

在项目根目录执行：

`python app.py`

然后在浏览器中访问 `http://127.0.0.1:5000/`。

操作步骤：

1. 在首页从歌曲列表中选择一首或多首你喜欢的歌曲。
2. 点击“生成推荐”按钮。
3. 在推荐结果页面中查看系统给出的相似歌曲列表。

## 代码结构说明

- `crawler.py`：示例爬虫 / 数据生成模块。项目运行时，如未发现 `data/songs.csv`，会自动生成一份示例数据。
- `recommender.py`：推荐算法模块，使用 TF-IDF + 余弦相似度做内容相似推荐。
- `app.py`：Flask Web 应用，提供简单的网页界面。
- `data/`：存放歌曲数据 `songs.csv`（首次运行时自动生成）。
- `templates/`：网页模板文件（首页和推荐结果页）。
